# Glade Product Roadmap

Welcome to the Glade Roadmap! Glade is currently in **Alpha**, and we have a lot of exciting features planned to make it the ultimate Markdown IDE.

## 🚧 Short-Term Goals (Alpha to Beta)
- **Plugin System:** Establish an API for developers to write custom plugins that integrate into the Milkdown editor and Tauri backend.
- **Enhanced Frontmatter Editing:** Improve the property pane to support specialized types (dates, tags, boolean checkboxes).
- **Global Search:** Full-text search across the entire vault using a local Rust indexer for extreme performance.
- **Distraction-Free Mode Enhancements:** Fullscreen integration, typewriter scrolling, and ambient themes.

## 🚀 Mid-Term Goals (Beta to v1.0)
- **Git Integration:** Built-in version control for vaults (commit, push, pull) directly from the sidebar.
- **Graph View:** A node-based visual representation of interlinked notes.
- **Multi-Tab Ergonomics:** Drag and drop tabs, split panes, and saved workspaces.
- **Cross-Platform Polish:** Ensure feature parity and native feel on macOS, Windows, and Linux.

## 🔮 Long-Term Vision
- **Mobile Companion Apps:** Native iOS/Android apps that sync with your desktop vaults via decentralized sync protocols.
- **Collaborative Editing:** Real-time peer-to-peer editing sessions.
- **Agentic AI Core:** Integrated Model Context Protocol (MCP) support for local LLMs to assist in writing, refactoring, and exploring knowledge.
